## Herramientas que necesitaremos instalar

* Cuenta de AWS
* AWS CLI
* NodeJS

## Objetivo

Tener un sitio web simulado de una tienda en linea.
El pago se realiza mediante paypal.

## Servicios que usaremos

* S3
* Lambda
* CloudWatch
* SNS
* DynamoDB
* ApiGateway
